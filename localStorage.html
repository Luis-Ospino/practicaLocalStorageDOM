Manipulación de datos con LocalStorage

LocalStorage es una herramienta compatible con HTML5 y los navegadores, gracias a esta podemos almacenar de manera local datos en el navegador que el usuario utiliza, mientras navega. Esta información quedará almacenada para que siempre que el usuario inicia en su navegador estos datos estén ahí. LocalStorage trabaja de manera similar que las cookies, sin embargo, cuenta con un espacio de 5Mb y al trabajar de manera local no necesita hacer peticiones al servidor.

¿Cómo se utiliza LocalStorage?

LocalStorage es una herramienta que se utiliza via JavaScript, donde podremos manipular datos almacenados de acuerdo a los códigos programados con este lenguaje. Para acceder a este almacenamiento se hace: Una vez ejecutada la aplicación --> click derecho --> inspeccionar --> application --> LocalStorage.

Métodos de LocalStorage para manipular datos

1. localStorage.setItem("clave",valor): este es el método para almacenar información en nuestro LocalStorage. En primer parámetro "clave" es donde ingresamos el nombre del elemento y en el parámetro valor ingresamos el dato que almacenará el parámetro clave.

2. localStorage.getItem("clave"): Este método sirve para traer el valor almacenado en una clave específica.

Intercambio de datos por medio de JSON

Para poder almacenar arreglos y objetos en el LocalStorage se deben de convertir en una cadena de caracteres(string), para eso usamos JSON.
